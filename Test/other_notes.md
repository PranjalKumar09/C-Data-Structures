### **Saving Space with `vector<int> count(3, 0)`**
**Fixed-Size Storage**: Uses a vector of size 3 to store counts of `'a'`, `'b'`, and `'c'` by mapping them to indices \(0, 1, 2\) using \(c - 'a'\), avoiding hash maps.
